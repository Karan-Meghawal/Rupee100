# Jamsrmlm

1. Install node modules

   ```shell
   npm i
   ```

1. Create a [.env](.env) file and add all fields of [.env.example](.env.example)

1. Push Mongodb models by prisma

   ```shell
   npx prisma db push
   ```

1. Start in dev mode

   ```shell
   npm run dev
   ```

1. Use `JFDT67GD` as License Code to install in admin panel
